# Risk Engine

A simple C++ portfolio risk simulator that models daily value changes based on market volatility and risk-free rates.

## Features

- Simulate portfolio value evolution over time
- Configurable market volatility and risk-free rate
- Support for multiple portfolios

## Build & Run

```bash
mkdir -p build && cd build
cmake ..
make
./riskEngine
```

## Requirements

- CMake 3.16+
- C++17 compatible compiler
